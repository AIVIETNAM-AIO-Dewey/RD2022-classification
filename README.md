# Image Preprocessing & Classification Project

Dự án nghiên cứu và thử nghiệm tác động của các kỹ thuật tiền xử lý ảnh (preprocessing) đến hiệu năng của mô hình mạng học sâu (Deep Learning / CNN) trong bài toán phân loại ảnh.

## 🚀 Bắt đầu nhanh (Quick Start)

Để thiết lập môi trường, tìm hiểu cấu trúc thư mục, quy trình chạy thực nghiệm và cách đóng góp vào dự án, vui lòng đọc:

👉 **[Hướng Dẫn Sử Dụng & Đóng Góp (CONTRIBUTING.md)](./CONTRIBUTING.md)**

## 📂 Tổng quan cấu trúc thư mục

*   `configs/`: Các tệp cấu hình tham số thực nghiệm.
*   `data/`: Thư mục chứa dữ liệu đầu vào (`raw/`, `processed/`, `external/`).
*   `notebooks/`: Jupyter Notebooks phục vụ EDA và thử nghiệm nhanh.
*   `outputs/`: Kết quả đầu ra (`models/`, `plots/`, `logs/`).
*   `src/`: Thư mục mã nguồn chính (`preprocessing/`, `models/`, `training/`, `evaluation/`).
*   `experiments/`: Kịch bản chạy thực nghiệm.
